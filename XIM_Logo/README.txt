XIMPLIC — LOGO ASSETS
=====================
Mark: "Core" · monochrome. Brand ink #141414, reverse #FFFFFF.

VECTOR (scales to any size — use these wherever possible)
  ximplic-mark.svg            Symbol, ink. Favicon / app icon / bullet.
  ximplic-mark-white.svg      Symbol, reversed (dark backgrounds).
  ximplic-lockup.svg          Symbol + "Ximplic" wordmark, ink.
  ximplic-lockup-white.svg    Lockup, reversed.
  favicon.svg                 Same as mark — modern browsers use this directly.

RASTER (transparent PNG — drop into slides, docs, anywhere SVG isn't accepted)
  png/ximplic-mark-{256,512,1024}.png            ink, transparent
  png/ximplic-mark-white-{512,1024}.png          reversed, transparent
  png/ximplic-lockup-black.png / -white.png      lockup, transparent, hi-res
  png/ximplic-lockup-black-1x.png / -white-1x.png  web-sized lockup

FAVICON / APP ICONS
  favicon/favicon-16.png, -32.png, -48.png       browser tabs (ink, transparent)
  favicon/apple-touch-icon.png  (180)            iOS home screen (ink tile, white mark)
  favicon/icon-192.png, icon-512.png             PWA / Android (maskable, ink tile)

WEB — paste into <head>:
  <link rel="icon" href="/favicon.svg" type="image/svg+xml">
  <link rel="icon" href="/favicon/favicon-32.png" sizes="32x32">
  <link rel="apple-touch-icon" href="/favicon/apple-touch-icon.png">

PRESENTATIONS
  PowerPoint 2016+ and Google Slides import SVG directly — use the lockup/mark SVG.
  Otherwise use the transparent PNGs. White versions are for dark slides.
  The wordmark is Space Grotesk (free, Google Fonts) — install it to edit the SVG text,
  or use the PNG lockups which need no font.

PREVIEW
  Open "Asset Sheet.html" to see every file rendered on light & dark.
